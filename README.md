# Dotsauce
Configuration for `*nix` systems. There's no place like `~/`.

## Installation
```sh
$ git clone https://github.com/yoshuawuyts/dotfiles.git src/yoshua/dotfiles
```
```sh
$ ./src/yoshua/dotfiles/script/bootstrap
```
